# Python packaging example

this is a sample of the basic use of `pyproject.toml` to develop and package a Python library

## See also

This is referred to from [MOOC "Python : des fondamentaux aux concepts avancés
du
langage"](https://www.fun-mooc.fr/en/cours/python-3-des-fondamentaux-aux-concepts-avances-du-langage/)
on France Université Numérique (FUN) platform  
and specifically in Week 6, Section 9, item " Comment organiser les sources de votre projet Python"

For a more complete description of packaging with `pyproject.toml`, see e.g.
<https://packaging.python.org/en/latest/guides/writing-pyproject-toml/>

## History

An initial version of this example was created before 2020 and was relying on `setup.py`.  
It has been re-written in 2024 to demonstrate the use of `pyproject.toml` for packaging.
